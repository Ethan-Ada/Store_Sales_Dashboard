Overview

This project is an interactive sales dashboard built using Power BI to explore sales performance across products, regions, and categories. The goal of this project was to practice data visualization and extract meaningful insights from a retail dataset.

Tools Used

- Microsoft Power BI
- Data Visualization
- DAX (basic measures) 


Features

- KPI cards showing total sales and total orders
- Sales trend over time (line chart)
- Sales by region comparison
- Top products by total sales
- Category and sub-category breakdown
- Region slicer for interactive filtering


Key Insights

- Sales are heavily concentrated in the East region, which significantly outperforms the other regions.
- Sales peaked in 2016, dropped in 2017, and recovered in 2018.
- Technology is the dominant category, generating most of the total sales.
- A few sub-categories (like accessories and machines) drive most of the revenue.
- A small number of products contribute a large portion of total sales.


What I Learned

- How to design a clean and interactive dashboard in Power BI
- How to use slicers to make reports dynamic
- How to structure visualizations to tell a clear data story
- How to identify key business insights from raw data


How to Use

1. Download the .pbix file
2. Open it in Power BI Desktop
3. Use the region slicer to explore the dashboard
